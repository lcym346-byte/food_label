商用工作流程

  1. 於「設定」建立公司／品牌／廠商資訊（一次設定，所有配方共用）。
  2. 建立或匯入原料營養資料。
  3. 新增配方並填寫「標示資訊」分頁（添加物、過敏原、有效日期等）。
  4. 產生 TFDA 九大項完整標示，列印或匯出。

設定 / 系統資訊

公司／廠商資訊

可建立多筆，配方標示時自下拉選單選用。也可在配方中手動覆寫單筆。

品牌資料

品牌可關聯到公司，方便同一家公司多品牌切換。

資料管理

列印與輸出

Web 版可使用瀏覽器列印；Android 版可透過系統列印服務輸出至熱感標籤機。標示字體已套用 ≥ 2mm 規範。

相容性

  • Web：Chrome / Edge / Safari / Android WebView。
  • Android：minSdk 21，Android 7.1.1(API 25) 可安裝。
  • 資料：IndexedDB／localStorage 離線保存。

新增原料

TFND/TFDA 離線營養資料庫查詢

輸入原料名稱後，優先比對內建台灣政府食品營養成分資料庫（TFND 2025 UPDATE1）；若連線可用，也會輔助查詢 Open Food Facts 與 USDA。

新增配方

配方原料

投入重量會自動換算成標示上的內容物排序（由高到低）。

產品基本

公司／廠商

覆寫本配方廠商資訊（不影響設定預設值）

有效日期 / 保存

食品添加物

系統會從原料名稱掃出候選（如「泡打粉」「鹽」「香料」），勾選後可選擇功能類別。亦可手動新增。

過敏原警語（TFDA 11 類）

系統會從原料偵測可能過敏原為「建議勾選」；請人工複核。

基因改造原料

含黃豆、玉米等基改可能原料須加註。

特殊警語 / 其他標示

例如：「苯酮尿症患者不宜食用」「本產品為重組肉」「開封後請冷藏」等。
